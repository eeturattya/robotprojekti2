At first only Google AI was used to swapping ideas for test plan and setting up coding environment. When the coding started same Google AI was used to help out with basic layout of Robot Framework. 
At later stages Github Copilot was activated for trial period and testcases + keywords started to get generated faster and faster.
In the end copilot was doing most of the writing and I was testing the flow and that testcases were working.
